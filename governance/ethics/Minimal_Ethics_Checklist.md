## Purpose

This checklist enables any researcher to conduct the Bootstrapped Pilot
ethically without a formal IRB, while maintaining standards consistent
with the Belmont Report and TLC governance requirements.

Complete all items before beginning Phase 2 (Pilot).
Sign and date. Publish to the repository before participant recruitment.

---

## Checklist

### Informed Consent

- [ ] Consent form written in plain language (max 8th grade reading level)
- [ ] Consent form reviewed by at least 1 advisory circle member
- [ ] Consent form explains: purpose, what data is collected, how it is used, how it is stored, right to withdraw
- [ ] Opt-out mechanism is immediate and requires no explanation
- [ ] Participants may withdraw at any time with no consequences
- [ ] Consent is obtained before any data collection begins
- [ ] Bootstrapped: consent recorded via Google Form with timestamp

### Data Protection

- [ ] No raw identifiable data leaves participant's browser
- [ ] Aggregate data only is stored (no individual identifiers)
- [ ] Data retention period stated explicitly (maximum: study duration + 2 years)
- [ ] Data deletion process described
- [ ] No third-party data sharing without explicit additional consent

### Participant Safety

- [ ] Sensory accommodation options available before session begins
- [ ] Universal "pause" button present and functional
- [ ] Cognitive load check-in at session midpoint
- [ ] Exit pathway is visible and accessible at all times
- [ ] No time pressure imposed (self-paced by default)
- [ ] Adversarial prompts reviewed by advisory circle before use
- [ ] Harm list reviewed and documented (instruments/chaos-tabletop-checklist.md)

### Neurodivergent-First Requirements

- [ ] Advisory circle has reviewed participant-facing materials
- [ ] High-contrast CSS and ARIA implemented
- [ ] Three interaction modes available (text, voice, visual icons)
- [ ] All instructions follow HCIP protocol (Article VII, Core Constitution)
- [ ] Feedback button ("How was this?") present and functional

### Researcher Conduct

- [ ] Researcher has completed minimal bias awareness review (30-min self-study)
- [ ] Member checking process planned (participants review interpretations)
- [ ] All conflicts of interest disclosed in published materials

### Documentation

- [ ] Signed checklist published to repository before recruitment begins
- [ ] Consent form version-controlled and published (redact participant names)
- [ ] Ethics checklist linked from OSF preregistration
