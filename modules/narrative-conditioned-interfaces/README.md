## What This Module Is

A TLC-governed research module producing an academic position paper arguing that
AI-augmented analytical tools must be redesigned around narrative-conditioned reasoning
to preserve human analytical fluency.

Core contribution: the **Narrative-First Adversarial Loop (NFL)** — a five-stage
protocol (Hypothesis → Evidence → Invariance → Adversarial Challenge → Refined Insight)
that structures AI-mediated data analysis to maintain intrinsic cognitive load.

---

## Why It Matters

As AI systems increasingly automate the analytical labor of data science, the bottleneck
shifts from computation to cognition. The risk — termed *Insight Atrophy* by industry
practitioners — is that repeated analytical offloading degrades users' capacity to
generate hypotheses, challenge conclusions, and transfer reasoning to new contexts.

This is not a proven causal claim. It is a **testable and designable concern** with
theoretical grounding in cognitive offloading literature, human-AI interaction research,
and sensemaking theory.
